# 货物进销管理系统简介 python116

## 项目概述

本项目是一款基于Python Web开发的货物进销管理系统，适用于中小型企业的库存管理。系统结合了Flask框架、MySQL数据库、JavaScript、HTML和CSS等技术，确保了系统的高效性和易用性。

## 技术栈

- **后端**: Python Flask框架
- **数据库**: MySQL
- **前端**: HTML, CSS, JavaScript, jQuery

## 功能模块

- 用户管理
- 商品管理
- 库存管理
- 销售管理
- 报表生成

## 系统角色

- 系统管理员
- 商品管理员
- 销售人员

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x
- 支持的浏览器：Chrome, Firefox, Safari等

## 部署步骤

1. 安装Python环境
2. 安装Flask框架
3. 配置MySQL数据库
4. 部署前端静态文件
5. 启动Flask应用

## 目录结构

```
- /static  # 存放静态文件如CSS、JS
- /templates  # 存放HTML模板文件
- /venv  # Python虚拟环境目录（可选）
- app.py  # Flask应用主程序
- models.py  # 数据模型定义
- ...
```
